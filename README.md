# cpp_course_advanced

Решения продвинутых домашних заданий по C++ в МФТИ (ФПМИ).

## Содержание
Проект содержит реализации кастомных аналогов стандартных контейнеров и утилит C++ с тестами:
- **StackAllocator** — кастомный аллокатор, работающий на заранее выделенной памяти (стеке/статическом буфере).
- **List** — двусвязный список, поддерживающий кастомные аллокаторы.
- **Deque** — двусторонняя очередь с итераторами произвольного доступа.
- **UnorderedMap** — хеш-таблица с открытой адресацией / списками коллизий.
- **Tuple** — кортеж с поддержкой `get`, `makeTuple` и `tupleCat`.
- **Smart Pointers** — собственные реализации `SharedPtr`, `WeakPtr` и механизма `EnableSharedFromThis`.
